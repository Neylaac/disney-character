## Projectbeschrijving
Disney Character Explorer is een interactieve single-page webapplicatie gemaakt voor het vak Web Advanced. De applicatie gebruikt de Disney API om Disney-personages op te halen en weer te geven.
Gebruikers kunnen Disney-personages bekijken, zoeken, filteren, sorteren en opslaan als favorieten. 
Het doel van dit project is om verschillende JavaScript-concepten uit de cursus Web Advanced toe te passen in één volledige interactieve webapplicatie.
